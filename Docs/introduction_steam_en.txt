RimPrison · Rebuilt

RimPrison provides a complete prisoner management system for your colony.
Main features:
Simple door access management
Dynamic economy system and prisoner thinking system
Prisoner schedules, labor, recreation, meals, wages, and debt system
Food and apparel whitelists with batch management
Prison suppression system, mental breaks, prison breaks, and turret deterrence rules

Early Release 1.0.6, supports backward compatibility with most features now complete. The current version is fully playable. However, it is still strongly recommended to save your game before using it. Compatible with the new "self-reliance" expansion(not published yet, old expansion version is not compatible).

Not compatible with the original RimPrison (Reset) mod — a fresh save is required. Any log messages containing "RimPrison.Reset" are caused by leftover data from the old version. Simply remove the old RimPrison (or old expansion) first (or start a brand new save).

Notice: This mod was created with AI assistance. Most code has been carefully reviewed by the developer(In EA version,some // [UNREVIEWED] comment was left in UI component and will be removed soon). This mod is open-sourced under the MIT license—everyone is welcome to review the code quality.

NOTICE: The "Door Access" feature is disabled by default in the mod settings BEFORE(fixed now). If you started a save before ea1.0.5, please enable it manually in the settings.

Main Features

💰 Dynamic Economy, Prisoner Thought System & Work System
· Full Work Compatibility: Prisoners can perform most types of work, including work added by other mods, with manually adjustable wages.
· Complete Dynamic Economy: A built-in economic system that dynamically adjusts based on prices and wages, covering prisoner labor, welfare, shopping, spending, and recreation.
· Prisoner Thought System: Prisoners dynamically make shopping decisions based on current prices / income / subsidies / management fees / shopping conditions and generate "thoughts"—observing your prisoners' thoughts helps you better understand how the economy is working.
· Debt Deduction via Organs: When wages cannot sustain a prisoner's living costs, organ deduction will automatically occur.
· Self-Ransom System: Once a prisoner's Edge Coins reach the set threshold, they can actively apply for self-ransom. Upon approval, they immediately become a guest and leave the map.
· Custom Currency Name: Freely name the currency for easy retrieval by tools like Rimtalk.

📋 Management & Interface

· Group Prisoner Management: Manage prisoner work / schedules / drugs / clothing / meals all in one place—no need to manage them individually.
· Detailed Logs: View clear logs of every prisoner's actions and all economic income and expenditure.
· Unique Suppression System: When properly managed and prisoners are in good condition, they will not trigger prison breaks or aggressive mental breaks.
· Selectable Management Styles: Different management approaches lead to different moods, event thoughts, and suppression effects.

🏠 Zone & Facility Recognition

· Prison Zones: Manually assign prison zones and prevent colonists from competing with prisoners for specific work within them.
· Simple Door Access: Provides a lightweight prisoner door access system, allowing prisoners to pass through areas they need without embedding an overly heavy system into the mod.
